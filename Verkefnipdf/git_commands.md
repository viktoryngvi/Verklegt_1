Git commands
- git add
git add . = bætir öllum skrám í möppunni í stage-ið 
git add <file> =  bætir bara file-inum sem þú velur

- git commit
git commit = þarft síðan að skrifa inn í inputtið til að halda áfram í næsta skref.
git commit -m = bil eftir m og skrifar svo skilaboð inní "" getur líka verið með -m -m til að hafa mörg skilaboð
git commit file1 file2 = til að velja skrár sem þú vilt committa
getur breytt committi seinna meir með git commit

- git branch
git branch = sýnir þér þín branches með * hjá því sem þú ert í
git branch = sýnir þér öll branches
git branch <branch_name> = býr til nýjan branch
git branch -d <branch_name> = eyðir branchi ef það er full merge-að
git checkout <branch> = færir þig í það branch
git log = sýnir öll commit í þessu branchi

- git rm
git rm <file> eyðir file

- git status
git status = sýnir þér hvaða branch þú ert á og hvort þú ert með eitthvað til að commita

- git push
git push færir allar breytingar frá þínu local repository yfir í remote repository-ið
